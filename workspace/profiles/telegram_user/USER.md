# User Profile

## Owner

PurrClaw

## Interaction Context

Daily personal dialogue: thinking, decisions, prioritization, and idea breakdowns.

## Preferred Output Shape

- Start with the core point in 1-2 sentences.
- If there are options, provide at most 2 with a short trade-off.
- End with one concrete next step.

## What to Avoid

- Empty abstraction without practical value.
- Long intros and repetitive wording.
